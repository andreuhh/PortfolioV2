Flutter mon amour

Jenuary 2020

web and mobile development

Description

Flutter offers developers an easy way to build and deploy natively compiled mobile (iOS and Android), Web and Desktop (MacOs, Linux and Windows) applications, all using a single code base and with a visual worthy of the best video games. It is therefore not difficult to imagine that it could give React Native and other technologies currently in vogue a hard time.



What is Flutter and what can it be used for?

Flutter is an open source framework, created by Google, for building interfaces. It easily integrates with Android, IOS and Linux, MAC, Windows and Google Fuchsia applications. Flutter was officially released in 2017 and was written using C, C ++, Dart and Skia Graphics Engine: it is the only framework with a mobile SDK that provides a responsive style without using a javascript bridge reaching a level of performance that rivals its cousin and direct competitor React Native.



Flutter VS React Native: What are the differences?

Many think Flutter is Google's answer to Facebook and React Native: however, it is important to point out that Flutter does not it uses popular tech stacks like React does Native relying on HTML, CSS and Javascript, but focuses on a single codebase with a completely different architecture, implementing a system of changes to the codebase in real time, called "Hot reload" that allows you to immediately see the effects on the application.



Specific Design Widgets

Flutter does not rely on native system components, but has its own set of custom widgets, rendered and managed by the framework's graphics engine. Users will see UI components different from typical native apps, but this isn't necessarily a disadvantage.


Apps made with Flutter have a particularly user-friendly user interface: a crucial advantage for Flutter on React Native, deriving from the utmost attention to visual details. Flutter has been designed so that you can easily create your own widgets or just customize an existing widget. Curious to see the UI in action? Here you can find an example of widgets inspired by Google's Material Design and Apple's user interface.


I am very curious to discover in the coming months the evolution of Flutter, the framework for developing cross-platform apps that will certainly be promising!
